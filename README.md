# IntroPython
Introduction to work with cxv files in python. Use of numpy, matplotlib and pandas libraries to manipulate  
Dataset link: https://data.wa.gov/Transportation/Electric-Vehicle-Population-Data/f6w7-q2d2 and visualize the 'Electric cars dataset.

-------
En este proyecto se desarrolla en Python un análisis básico de datos sobre Vehiculos Electricos vendidos en los estados Unidos. Los datos provienen de https://data.wa.gov/Transportation/Electric-Vehicle-Population-Data/f6w7-q2d2 . A partir de ellos he realizado varias exploraciones y visualizaciones de autos vendidos por estado, tecnologias o marcas.
